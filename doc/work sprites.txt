* Importar modelo a Blender
* Quedarse sólo con la geometría necesaria.
* Texturizar, iluminar, generar UVs
* Cocinar textura
* Exportar a OBJ
* Usar https://www.drububu.com/miscellaneous/voxelizer/?out=kvx para generar voxels. Exportar como VOX o KVX
* Usar Goxel o MagicaVoxel para generar slices
* Dividir slices en imágenes sueltas con: magick convert slices.png -crop NUMXxNUMY@ +repage +adjoin %03d.png
* Usar el editor de FPG de BennuGD para generar un FPG con las slices
